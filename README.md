# DedSec — Agentic AI Cybersecurity System

> "We Are Many. We Are DedSec."

DedSec is a dual-engine agentic AI cybersecurity 
system built to stay alive even under attack. 
Inspired by aeronautical redundancy standards — 
if one engine fails, the other takes over instantly.

## Architecture

**Core Engine** — Primary agentic brain, processes 
all commands and decisions

**Aegis** — Shadow engine that activates on primary 
system compromise, isolates the attack, counter-attacks 
the hacker's network and retrieves the main engine

**Sentinel-V1** — Heartbeat watchdog that monitors 
all system modules for anomalous behavior and triggers 
automated failover response

## Modes
- Assistant Mode
- Recon Mode  
- Pentest Mode
- Forensic Mode
- Red Team Mode
